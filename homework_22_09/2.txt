Фундаментальные типы данных:

int	- 4 bytes - Выбор по умолчанию для целых чисел.
double - 8 bytes - Выбор по умолчанию для чисел с плавающей точкой.
bool - 1 byte - true/false. Работа с булевым типом данных.
char - 1 byte - ASCII-символы в старом C-style или объект std::string, который никогда не будет в кодировке Unicode.
wchar_t - 2 bytes - Представляет "расширенные" значения символов. Это тип символов, который используется в строках типа std::wstring.
unsigned char - 1 byte - Отражает байтовое значение от 0 до 255
unsigned int - 4 bytes - Выбор по умолчанию для битовых флагов.
long long - 8 bytes - Представляет очень большие целочисленные значения.
void type - 0 bytes - Это особый тип; нельзя объявить переменную типа void, но можно объявить переменную типа void* (указатель на void), что иногда необходимо при выделении сырой (нетипизированной) памяти
